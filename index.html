AVAILABLE POSITIONS

& what we'll need from you

Web Developer

Your primary task as a web developer will be to create reliable and high-performing applications and services, which can be accessed over the internet. Focusing solely on the underlying software and databases (known as the 'back end') is most common, however some web developers work on the interface and visual design for us, (the 'front end'), we also employ others who combine both ('full-stack development').


Your responsibilities in the role:

  • Write code in one or more programming or scripting languages, such as PHP or JavaScript
  • Plan and prototype new applications
  • Design the architecture of the components of an application
  • Build and test Application Program Interfaces (APIs) for applications to exchange data
  • Stay up to date with new trends and advancements in web development build and maintain databases
  • Meet designers, developers and project staff for progress updates

You'll typically work normal office hours, 9am to 5pm, Monday to Friday. However as previously stated we are flexible and allow for personal work hours if needed. And work can be completed from the comfort of your own home if prefered (you will be required to attend faculty meetings in person from time to time.)
There is no formal qualification needed to become a web developer in the UK, though some employers prefer applicants to have a technical degree. Relevant subjects include:
computer science
informatics
software engineering
web design and development.
Knowledge and experience are usually the most important requirements. A relevant degree can provide you with demonstrable knowledge, particularly the fundamental principles of programming and application design.
